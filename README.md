# Teslo API

1. clonar el proyecto
2. ```yarn install```
3. Clonar el archibo ```.env.template``` y renombrarlo a ```.env```
4. cambiar las variable de entorno
5. levantar la base de datos
```
docker-compose up -d
```
6. levantar ``` yarn start:dev```
